#  Chatroom (C++ & Epoll & Redis)

一个简易的聊天室

## 编译方式

```bash
- 进入chatroom目录
- mkdir build
- cd build
- cmake ..
- make
```

会生成：
- `client`：客户端可执行文件
- `server`：chat 服务端可执行文件
- `FTP_serve`：FTP服务端可执行文件
